# 🎬 Movie Search App

A responsive Movie Search application built with **HTML, CSS, and JavaScript** that allows users to search for movies using the **OMDb API**.

## ✨ Features

- 🔍 Search movies by title
- 🖼️ Display movie posters
- 🎞️ Show movie title and release year
- ⌨️ Press **Enter** to search
- ❌ Error handling for invalid movie names
- 📱 Simple and responsive user interface

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- OMDb API
